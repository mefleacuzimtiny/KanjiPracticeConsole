# KanjiLearnConsole
A Japanese learning tool that prints random kanji characters and reads them using Text-to-Speech. The aim is to write them in the correct stroke order on a notebook alongside and then check the answer with the screen. The entire program works directly in the console.

## Usage Instructions:
1. Open the Command Prompt on Windows.
2. Then install the JamDict and PyTTSX3 libraries using the following commands:
```cmd
pip install pyttsx3
pip install jamdict
```

## Dependencies
As mentioned above, the program uses the following dependencies.
https://pypi.org/project/jamdict/
