# KanjiLearnConsole
A Japanese learning tool that prints random kanji characters and reads them using Text-to-Speech. The aim is to write them in the correct stroke order in a notebook and then check the answer on the screen. The entire program works directly in the console.

## Usage Instructions
1. Open Command Prompt on Windows.
2. Install the required libraries:
```cmd
pip install pyttsx3
pip install jamdict
```

## Dependencies
The program uses the following libraries:

- JamDict: <https://pypi.org/project/jamdict/>
- pyttsx3: <https://pypi.org/project/pyttsx3/>
